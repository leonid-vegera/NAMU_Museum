# Museum landing page

Responsive page for different screen sizes:
- Large screens 2560px
- Full HD 1920px
- The design 1600px
- Notebook 1280px
- Tablet 1024
- Mobile (> 320px)

Technologies used: HTML, CSS, JavaScript
